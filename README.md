## Run locally

Prerequisites: **Git**, **Node**, **Yarn**

1. Clone the github repository using the Github CLI.

```sh
git clone https://github.com/internnova/internnova
```

2. Change to the project directory.

```sh
cd internnova
```

3. Installation commands

```sh
yarn install
```

5. Run

```sh
yarn run dev
```
